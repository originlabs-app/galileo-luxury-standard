## 1. Typography Refonte — "Swiss-Luxe" Stack

**Rationale:** L'interopérabilité en 2026 passe par une lecture ultra-claire. Le duo néo-grotesque/géométrique remplace Inter.

### New Font Stack

| Role | Font | Characteristics |
|------|------|-----------------|
| UI / Body | **Satoshi** (Variable) | Néo-grotesque, précision chirurgicale |
| Titles (h1, h2) | **Outfit** | Géométrique, "tech-forward" |
| Heritage accents | **Cormorant Garamond** (Italic) | Manuscrit/artisanal (citations, labels) |

### CSS Variables

```css
--font-satoshi: 'Satoshi', sans-serif;
--font-outfit: 'Outfit', sans-serif;
--font-serif: 'Cormorant Garamond', serif;

h1, h2 { font-family: var(--font-outfit); font-weight: 600; }
body { font-family: var(--font-satoshi); }
.italic-accent { font-family: var(--font-serif); font-style: italic; }
```

### Font Sources

- **Satoshi:** https://www.fontshare.com/fonts/satoshi (free, variable)
- **Outfit:** Google Fonts (free, variable)
- **Cormorant Garamond:** Already in project

## 2. Mesh Gradients "Deep Space"

**Rationale:** Au lieu du fond noir uni (#050505), injecter une profondeur atmosphérique avec des nuages de couleurs très sombres qui semblent flotter loin derrière l'interface.

### New Palette

```css
--obsidian: #050505;          /* Base (unchanged) */
--deep-plum: #1A0B2E;         /* New: Purple depth */
--midnight-blue: #020617;      /* New: Blue depth */
--precision-blue: #00A3FF;     /* Existing accent */
```

### Spatial Background Implementation

```css
.spatial-background {
  background-color: #050505;
  background-image:
    radial-gradient(at 0% 0%, rgba(26, 11, 46, 0.15) 0px, transparent 50%),
    radial-gradient(at 100% 100%, rgba(0, 163, 255, 0.05) 0px, transparent 50%);
}
```

### Considerations

- Keep `@media (prefers-reduced-motion)` support
- Consider CSS `color-mix()` for dynamic gradients
- Test performance on mobile (gradient layers)

## 3. Angle Glow Borders

**Concept:** Au lieu d'une bordure uniforme, seule l'arête qui "capte la lumière" s'allume. La bordure disparaît sur les segments droits et brille uniquement dans les coins lors du survol.

### Implementation Approaches

**Option A: Conic Gradient Border**
```css
.angle-glow {
  --glow-color: var(--gold);
  border: 1px solid transparent;
  background:
    linear-gradient(var(--surface), var(--surface)) padding-box,
    conic-gradient(
      from 45deg,
      transparent 0deg,
      var(--glow-color) 45deg,
      transparent 90deg,
      transparent 180deg,
      var(--glow-color) 225deg,
      transparent 270deg
    ) border-box;
  border-radius: 12px;
}
```

**Option B: Pseudo-element Masks**
```css
.angle-glow::before {
  content: '';
  position: absolute;
  inset: -1px;
  border-radius: inherit;
  background: conic-gradient(from 45deg, transparent, var(--gold), transparent);
  mask: linear-gradient(#fff 0 0) content-box, linear-gradient(#fff 0 0);
  mask-composite: exclude;
  padding: 1px;
  opacity: 0;
  transition: opacity 0.3s;
}
.angle-glow:hover::before {
  opacity: 1;
}
```

### Animation Enhancement

- Add subtle rotation on hover (45deg → 90deg)
- Consider `animation: glow-rotate 4s linear infinite` for ambient version

## 4. Bento Grid Layout (Features Section)

**Current:** Equal-width grid
**Proposed:** Asymmetric layout that breaks monotony

### Layout Structure

```
+-----------------------+-----------------------+
|  [ CODE: JSON-LD ]    |  [ STAT: 1.2M ]       |
|  (Grande carte 2/3)   |  (Petite carte 1/3)   |
+-----------------------+-----------------------+
|  [ ICON: SECURITY ]   |  [ CODE: SOLIDITY ]   |
|  (Petite carte 1/3)   |  (Grande carte 2/3)   |
+-----------------------+-----------------------+
```

### Grid CSS

```css
.bento-grid {
  display: grid;
  grid-template-columns: 2fr 1fr;
  grid-template-rows: auto auto;
  gap: 1.5rem;
}

.bento-grid > :nth-child(3) {
  grid-column: 1;
}

.bento-grid > :nth-child(4) {
  grid-column: 2;
}

/* Alternate row: swap proportions */
.bento-grid > :nth-child(3),
.bento-grid > :nth-child(4) {
  /* Handled by explicit column placement */
}
```

### Liquid Glass Effect

Each Bento block has a reflective shine that follows the mouse:

```css
.liquid-glass {
  position: relative;
  overflow: hidden;
}

.liquid-glass::after {
  content: '';
  position: absolute;
  inset: 0;
  background: radial-gradient(
    circle at var(--mouse-x, 50%) var(--mouse-y, 50%),
    rgba(255, 255, 255, 0.08) 0%,
    transparent 50%
  );
  pointer-events: none;
}
```

```typescript
// Mouse tracking (client component)
function handleMouseMove(e: React.MouseEvent) {
  const rect = e.currentTarget.getBoundingClientRect();
  const x = ((e.clientX - rect.left) / rect.width) * 100;
  const y = ((e.clientY - rect.top) / rect.height) * 100;
  e.currentTarget.style.setProperty('--mouse-x', `${x}%`);
  e.currentTarget.style.setProperty('--mouse-y', `${y}%`);
}
```

## 5. Hero Audit — Spatial Redesign

### Current Issues

- Everything centered (2023 pattern)
- Static composition
- No dimensional depth

### Proposed Changes

**A. Asymmetric Layout**
- Text block LEFT (60% width)
- Floating architecture diagram RIGHT (40%)
- Diagram: semi-transparent, subtle parallax or float animation

**B. Micro-Haptic Feedback**
```css
.button-haptic {
  transition: transform 0.1s ease-out;
}

.button-haptic:active {
  transform: scale(0.98);
}
```

**C. Title Treatment**
- Larger, bolder Outfit font
- Gradient text (gold → white) for emphasis
- Subtle text-shadow for depth

### Layout Structure

```
+----------------------------------+------------------+
|  [SMALL BADGE: v1.0.0]           |                  |
|                                  |   [FLOATING      |
|  GALILEO LUXURY                  |    ARCHITECTURE  |
|  STANDARD                        |    DIAGRAM]      |
|                                  |                  |
|  [Tagline text, left-aligned]    |   (3D/Parallax   |
|                                  |    effect)       |
|  [CTA Buttons, left-aligned]     |                  |
+----------------------------------+------------------+
```

## 8. Performance Considerations

- **Fonts:** Use `font-display: swap` for Satoshi/Outfit
- **Gradients:** CSS-only, no performance impact
- **Liquid Glass:** `pointer-events: none` on overlay, minimal JS
- **Reduced Motion:** Disable animations for `prefers-reduced-motion`
