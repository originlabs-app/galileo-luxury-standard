# Changesets

Hello and welcome! This folder has been automatically generated by `@changesets/cli`, a build tool that works
with multi-package repos, or single-package repos to help you version and publish your code. You can
find the documentation for changesets in [our docs](https://github.com/changesets/changesets/).

We have a list of [common questions](https://github.com/changesets/changesets/blob/main/docs/common-questions.md) - please read these! There may be information in there that you need.

We also have answers to [questions about the preconstruct tool](https://github.com/preconstruct/preconstruct/blob/main/docs/faq.md) if you're using that.

If you're unsure what this folder is for, you may want to delete it and let changesets itself recreate it next time you want to add a changeset.
